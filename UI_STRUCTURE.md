- **Back Button**: Returns to grid view
- **Previous/Next**: Cycle through photos
- **Zoom**: Mouse wheel or pinch gesture
- **Pan**: Drag when zoomed

- **Background**: ApplicationPageBackgroundThemeBrush
- **Cards**: CardBackgroundFillColorDefaultBrush

## Responsive Behavior

### Grid Layout
- **Auto-Wrapping**: Items wrap based on window width
- **Flexible Columns**: Calculates columns dynamically
- **Maintains Aspect**: Cards keep 280px width
- **Vertical Scroll**: ScrollViewer handles overflow

### Window Sizes
- **Minimum**: Supports down to 800px width
- **Optimal**: 1400Ã—900 pixels (default)
- **Maximum**: Unlimited, scales up

## Accessibility Features

- **Keyboard Navigation**: Full keyboard support
- **Screen Reader**: XAML structure supports narration
- **High Contrast**: Respects system theme
- **Focus Indicators**: Clear focus states
- **Touch Friendly**: 280px cards suitable for touch
